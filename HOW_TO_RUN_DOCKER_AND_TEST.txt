1. Open a terminal in the project root:
   cd loan-platform-suite

2. Build and start the backend with Docker Compose:
   docker compose up --build

3. Check these URLs in your browser:
   API health:        http://localhost:3000/health
   Swagger UI:        http://localhost:3000/api-docs
   Raw Swagger file:  http://localhost:3000/openapi.yaml
   Agent health:      http://localhost:4000/health
   Agent root:        http://localhost:4000/

4. Frontend local run:
   cd frontend
   npm install
   npm run dev

5. Open the frontend in the browser:
   http://localhost:5173

6. Login using the hardcoded credentials:
   Username: demo.user
   Password: Xgen@123

7. In the conversation, enter only this field:
   Swagger URL: http://localhost:3000/openapi.yaml

8. Then send a prompt like:
   Evaluate the loan application workflow and explain the results in simple language.

9. Screenshots to send:
   - docker compose terminal showing both containers started
   - http://localhost:3000/api-docs
   - http://localhost:4000/health
   - the login page
   - the conversation page with the Swagger URL field
   - a successful result after sending a prompt

10. Optional Postman test:
   POST http://localhost:4000/qa/run
   Body JSON:
   {
     "prompt": "Evaluate the loan application workflow and explain the results in simple language.",
     "swaggerUrl": "http://localhost:3000/openapi.yaml"
   }
